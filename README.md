# ChimeraKey
Инструмент для защиты от поведенческой биометрии. Скрывает клавиатурный почерк и движения мыши от сайтов и приложений.


# ChimeraKey v0.9 (Demo)

**Инструмент для защиты от поведенческой биометрии в Windows.**

ChimeraKey — это приложение с открытым исходным кодом, которое искажает ваш клавиатурный почерк и скрывает движения мыши от веб-сайтов и приложений, делая вас менее узнаваемым для систем поведенческого анализа.

> **Демо-версия:** Проект находится в стадии активной разработки. Текущая версия является демонстрационной и содержит известные ограничения (см. ниже).

---

## Возможности

### Клавиатурный камуфляж
- Подмена таймингов нажатий (задержки, длительность удержания клавиш)
- Случайные опечатки с имитацией человеческого поведения
- Поддержка профилей морфинга (новичок, профи, хаотичный)
- Корректная работа с модификаторами (Shift, Ctrl, Alt)

### Ghost Mouse (невидимка)
- **Движения мыши скрываются** — приложения не получают события `WM_MOUSEMOVE`
- **Клики эмулируются** с небольшой случайной задержкой через абсолютные координаты
- **Системный курсор скрыт(+-)**, вместо него отображается фальшивый крестик (overlay)
- **Колёсико работает в штатном режиме** без задержек

### Проверка целостности
- Отслеживание изменений всех файлов проекта
- Предупреждение при модификации (но не блокировка функционала)
- Эталонные хеши хранятся в зашифрованном виде

---

## Известные ограничения

### Ghost Mouse
- **Мерцание hover-эффектов** на сайтах — из-за того, что браузеры могут получать координаты курсора через `GetCursorPos()` и Raw Input в обход хуков. Полностью решается только DLL-инъекцией или драйвером (планируется в следующих версиях).
- **Drag-and-drop не работает** при активном Ghost Mouse — временно отключайте его для перетаскивания.
- **Сайты с активным использованием Raw Input** (например, Google Maps) могут частично отслеживать движения.
